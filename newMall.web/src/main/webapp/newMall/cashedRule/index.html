• 即日起，每人均可在小程序中将余额提现至微信零钱，服务费均按提现金额的0.9%（千分之九）收取，提现金额需超过1元以上。

  • 规则说明：

    • 1、提现金额只能提现到打开小程序的打钱微信账户中，为防止网络金融诈骗，不允许用户提现至指定微信余额，如遇特殊情况，可打客服电话向平台申请.

    2、提现金额需大于1元以上，每笔提现服务费均按提现金额的0.9%（千分之九）收取,服务费的计算精确到小数点后两位，例如：提现169元，则收取服务费0.17元.

    3、若需申请提现服务费发票，可打客服电话向平台进行申请.

    4、提现金额在3个工作日内到达微信余额中，请及时查看.

    5、提现服务费将从用户余额中额外扣除.

  • 关于提现的问题，可以在【新商城】小程序中，【我的】页面中，联系【我的客服】；

  • 在法律法规许可的范围内，【新商场】拥有本活动的最终解释权；

  • 该活动与微信官方无关，与手机生产商如苹果公司无关；
